## Loan Analytics Dashboard | Power BI Project
An interactive Power BI dashboard project designed to analyze loan distribution, customer behavior, risk analysis, and profitability insights for financial decision-making.

## This project helps understand:

-Loan performance
-Customer loan behavior
-Risk distribution
-Profitability trends
-Verification impact on payments
-State-wise and grade-wise loan analysis
## Dashboard Overview
## Dashboard 1: Overview & Customer Insights
This dashboard focuses on customer analysis and overall loan distribution.

## Key Features
-Total Loan Amount Analysis
-Total Customers Overview
-Total Payment Received
-Average DTI (Debt-to-Income Ratio)
-Average Interest Rate
-Loan Amount vs Payment by Grade
-Loan Amount by Term (36 vs 60 months)
-Loan Amount by Home Ownership
-Loan Amount by Employment Length
-Loan Amount by Verification Status
-DTI vs Loan Amount Analysis
-Top States by Loan Amount
## Key Insights
-Majority of loans are issued for 36-month terms
-Mortgage holders contribute the highest loan amounts
--Grade B customers generate the highest loan value
-Higher employment length is associated with larger loan amounts
-Verified customers show stronger payment behavior

## Dashboard 2: Risk & Profitability Insights

This dashboard focuses on risk evaluation and business profitability analysis.

## Key Features
-Interest Rate by Grade
-Profit by Grade
-Risk Distribution Analysis
-Total Profit KPI
-Loan Application by Purpose
-Verification Status Impact on Payment
-Loan Amount by Loan Term
-Risk vs Profit Insights

## Key Insights
-Grade B and C loans generate the highest profits
-High-risk loans dominate the portfolio
-Debt consolidation is the most common loan purpose
-Verified loans receive higher payments compared to non-verified loans
-36-month loans contribute higher repayment amounts
